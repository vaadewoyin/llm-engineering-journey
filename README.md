## 📌 Overview

This repository documents a structured 13-week journey into LLM engineering, focusing on building practical skills in:

- **Foundations**: PyTorch, attention mechanisms, transformers
- **Fine‑tuning**: LoRA, QLoRA, DPO, preference alignment
- **Retrieval**: vector databases, RAG pipelines
- **Production & MLOps**: Serving, observability, CI/CD, containerization
